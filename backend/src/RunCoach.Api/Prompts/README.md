guardrails that make RunCoach what it is. The Apache-2.0 code is a generic
deterministic-plus-LLM framework and belongs to the OSS commons; the prompts
are the product-specific instructions and are held under a share-alike,
non-commercial license so the coaching IP cannot be lifted wholesale into a
