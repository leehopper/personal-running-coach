### Key findings

- **Token budget:** 12K recommended as default (actual usage peaks at ~4,500 tokens total including system prompt)
- **Positional placement:** Profile-at-start confirmed as optimal (zero token cost, enables caching, aligns with U-curve research)
- **Summarization:** Mixed mode (L1 recent + L2 older) is the best balance of detail and efficiency
- **Conversation history:** 5 turns adds ~785 tokens, well within budget; supports 0-10 turns
- **Cost estimate:** ~$0.044 per API request; ~$2.64-6.60/month per active user depending on usage pattern
