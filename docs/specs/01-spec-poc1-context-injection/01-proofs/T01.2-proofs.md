## Formula Details

Uses the Daniels/Gilbert oxygen cost equations (Jack Daniels, "Daniels' Running Formula"):

- **Oxygen cost:** VO2 = -4.60 + 0.182258 * v + 0.000104 * v^2 (v = meters/minute)
- **Fractional utilization:** %VO2max = 0.8 + 0.1894393 * e^(-0.012778 * t) + 0.2989558 * e^(-0.1932605 * t) (t = minutes)
- **VDOT:** VO2 / %VO2max

## Design Decisions

- Used `FrozenDictionary` for distance lookup (immutable, optimized for read-heavy access)
- Case-insensitive string comparison via `StringComparer.OrdinalIgnoreCase`
- Multiple race times: returns highest VDOT (best performance = most representative fitness)
- Returns `decimal?` to handle null cases (no race history, unsupported distances)
- Rounded to 1 decimal place for practical use
