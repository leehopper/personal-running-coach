## Design Decisions

- Profiles placed in `src/RunCoach.Api/Modules/Training/Profiles/` (not test project) so they are accessible from both the test project and the future console app
- VDOT and paces computed dynamically from VdotCalculator and PaceCalculator (not hardcoded) so changes to the calculators automatically propagate
- TestProfile record type in its own file per SA1402 (one type per file)
- Training history uses realistic workout patterns matching each profile's constraints
- James's workouts are all Easy type, none exceeding 20 minutes (matching physiotherapist clearance)
- Priya's history has exactly 4 workouts per week (matching her constraint)
- All dates use deterministic values (today = 2026-03-21) for reproducible tests
