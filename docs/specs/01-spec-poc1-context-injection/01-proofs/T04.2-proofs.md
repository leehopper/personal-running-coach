### LLM live run readiness

The infrastructure fully supports live LLM runs. Transition from dry-run to live:
1. Configure API key: `dotnet user-secrets set "Anthropic:ApiKey" "<key>" --project src/RunCoach.Poc1.Console`
2. Create executor with LLM: `new ExperimentExecutor(outputDir, llmClient)`
3. Run live: `await executor.RunAllExperimentsAsync(live: true)`

Results will include `llmResponse` and `hasJsonPlan` fields for qualitative analysis.
