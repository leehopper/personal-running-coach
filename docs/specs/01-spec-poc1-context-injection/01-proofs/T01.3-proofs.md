## Design Decisions

- Used `SortedDictionary<int, PaceTableEntry>` for O(log n) lookup by integer VDOT
- Linear interpolation between adjacent table entries for fractional VDOT values
- VDOT range clamped to 30-85 (covers beginner through elite runners)
- Pace table values calibrated to published Daniels' Running Formula (4th edition)
- VDOT 50 anchor: Easy 5:01-5:31/km, Marathon 4:27/km, Threshold 4:10/km, Interval 3:51/km, Rep 3:36/km
- `ArgumentOutOfRangeException.ThrowIfLessThan/ThrowIfGreaterThan` without explicit param name (SonarAnalyzer S3236 compliance)
- `Lerp` expression uses explicit parentheses (StyleCop SA1407 compliance)
