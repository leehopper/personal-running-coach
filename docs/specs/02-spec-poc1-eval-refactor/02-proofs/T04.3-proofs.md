## What Was Done

### Critical Fix: AnthropicStructuredOutputClient DelegatingChatClient
The Anthropic SDK's `AsIChatClient()` bridge does NOT translate `ChatResponseFormat.ForJsonSchema()` to constrained decoding — it silently ignores the schema. Created a custom `DelegatingChatClient` wrapper that intercepts structured output requests and delegates to the native Anthropic SDK's `OutputConfig.JsonOutputFormat`. All calls (structured + unstructured) flow through a single cached IChatClient pipeline.

### Model ID Fix: Floating Aliases
Changed from hardcoded dated model IDs to floating aliases:
- `claude-sonnet-4-6` for coaching (supports structured output)
- `claude-haiku-4-5` for LLM-as-judge
The old `claude-sonnet-4-20250514` (Sonnet 4.0) does not support structured output at all.

### Constraint Relaxation: Injured Profile
Relaxed injured workout type constraint from "Easy only" to "Easy, LongRun, or Recovery" — a long easy run at controlled pace is a legitimate recovery workout.
