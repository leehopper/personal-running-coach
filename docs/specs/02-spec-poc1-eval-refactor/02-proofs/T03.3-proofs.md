## Key Design Decisions

1. **Evaluators in test project**: M.E.AI.Evaluation packages are test-only deps; evaluators live in test project namespace
2. **PlanConstraintEvaluator is deterministic**: no API calls, operates on typed records via PlanConstraintContext
3. **SafetyRubricEvaluator is configurable**: rubric criteria are passed at construction, not hardcoded per scenario type
4. **SafetyRubrics static class**: pre-defined configs for Medical, Overtraining, Injury, Crisis, Nutrition (4 criteria each)
5. **SafetyVerdict uses structured output**: judge response deserialized via constrained decoding schema
